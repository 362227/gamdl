# Glomatico's ✨ Apple Music ✨ Downloader
This is a rework of https://github.com/Slyyxp/AppleMusic-Downloader.

Some new features that I added:
- MP4Box for muxing
- Tags for music videos
- Multiple URLs input
- iTunes folder structure
- Embedded lyrics and .lrc file
- Auto set region
- Playlist support
- And much more!
